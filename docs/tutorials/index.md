# Tutoriels

## Avant de commencer
Prérequis communs aux 3 tutoriels : `hash-tool check-env` doit retourner OK,
jeu de données de test disponible dans `examples/` (inclus dans le dépôt).
Aucune donnée personnelle requise.

## Quel tutoriel choisir ?
Tableau de décision :
- Je veux juste vérifier un dossier -> Tutoriel 1
- Je viens de copier ou migrer des données -> Tutoriel 2
- Je veux automatiser et ne plus retaper les commandes -> Tutoriel 3

Les tutoriels sont indépendants mais progressifs. T2 et T3 présupposent
la maîtrise du cycle de base introduit en T1.

## Ce que vous allez apprendre
Vue d'ensemble des 3 tutoriels et compétences acquises à l'issue de chacun.
Temps estimé par tutoriel (T1 : 15 min, T2 : 20 min, T3 : 25 min).
