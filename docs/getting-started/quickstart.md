# Démarrage rapide

## Premier compute
Calcul d'empreintes sur un dossier en une commande. Fichier `.b3` et sidecar
`.meta.json` produits. Explication de ce que contient chaque fichier.
Durée estimée pour 1000 fichiers.

## Premier verify
Vérification immédiate après compute sur le même dossier. Sortie nominale
(tout OK) montrée. Signification du code de sortie `0`.

## Premier compare
Comparaison de deux bases existantes. Les 5 fichiers de résultats produits
listés avec leur rôle. Ouverture du `report.html` dans un navigateur.
Lien vers les tutoriels pour aller plus loin.
