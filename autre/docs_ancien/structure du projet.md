

# Structure des Fichiers

Voici l'organisation simplifiée du projet :

* **src/** : Contient toute la logique du programme.
* `setup.sh` : **Fichier unique de configuration** (chemins source et destination).
* `main.sh` : **Lanceur principal** (exécute tout le processus).
* 
`collect.sh` : Gère le hachage et la normalisation.


* 
`report_gen.py` : Génère le rapport HTML final.




* **test/** : Dossiers de données et résultats.
* 
`source/` & `destination/` : Vos fichiers à comparer.


* 
`hash/` : Contient le fichier `differences.txt` et le `rapport.html` final.
